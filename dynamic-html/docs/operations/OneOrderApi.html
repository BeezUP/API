This is the API

changeOrder

URL

https://api.beezup.com/v2/user/marketplaces/orders/{marketplaceTechnicalCode}/{accountId}/{beezUPOrderId}/{changeOrderType}

HTTP Method

POST

Response Type

changeOrderResponse

Parameters

  • Path: marketplaceTechnicalCode String

    The marketplace technical code

  • Path: accountId Integer

    The account identifier

  • Path: beezUPOrderId String

    The order BeezUP identifier

  • Path: changeOrderType String

    The order change type

  • Query: userName String

    Sometimes the user in the e-commerce application is not the same than the subscription key you indicate in your settings. We recommand you to indicate the login of the user in your appplication.

  • Body: request ChangeOrderRequest(ChangeOrderRequest)

  • Query: testMode Boolean

    If true, the operation will be be commited. But the validation will be taken in account.
